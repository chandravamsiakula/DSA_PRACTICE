# Valid Majority Element

[![Difficulty](https://img.shields.io/badge/Difficulty-Easy-brightgreen)](https://leetcode.com/problems/majority-element/description/)

---

A majority element is an element that appears more than ⌊n / 2⌋ times in an array.

Given an integer array `nums`, return the majority element.

You may assume that the majority element always exists in the array.

---

## Example 1:

```bash
Input: nums = [3,2,3]
Output: 3
Explanation: 3 appears more than ⌊3 / 2⌋ = 1 time.
```

---

## Example 2:

```bash
Input: nums = [2,2,1,1,1,2,2]
Output: 2
Explanation: 2 appears more than ⌊7 / 2⌋ = 3 times.
```

---

## Constraints:

- `1 <= nums.length <= 5 * 10^4`
- `-10^9 <= nums[i] <= 10^9`
